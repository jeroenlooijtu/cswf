id Functional requirement Acceptatie criteria Prioriteit
F-01 Een gebruiker moet een oefening kunnen invullen en opslaan Bij een oefening moet de gebruiker der hoeveelheid gewicht gebruikt en het aantal repetities uitgevoerd kunnen invullen. Ook kan de gebruiker optioneel de rpe(Rate of perceived exertion) invullen
F-02 Een gebruiker moet de oefeningen later terug bekijken in een lijst - Deze lijst kan gefilterd worden op oefening zodat de gebruiker kan zien hoe zijn of haar performance over tijd is veranderd
- De naam van de oefening, het gebruikte gewicht, de datum van de oefening en de rpe moeten zichtbaar zijn.
- De lijst moet gesorteerd kunnen worden op datum en op gewicht
- Bij sorteren op gewicht moeten entry's met gelijk gewicht aflopend gesorteerd worden op aantal repetities
F-03 Een gebruiker kan een workout aanmaken - Een workout bestaat uit oefeningen aangeleverd door de applicatie
- De gebruiker bepaalt de volgorde van de oefeningen in de workout
- De gebruiker geeft aan hoeveel sets van een oefening gedaan worden
F-04 Een gebruiker kan een lijst met workouts bekijken - De lijst laat de naam van de workout zien
- De lijst laat de tijd van de workout zien
- De lijst bestaat uit eigen gemaakte workouts en workouts van andere gebruikers die de gebruiker zelf heeft toegevoegd
F-05 Een gebruiker kan een trainingsplan maken - Dit trainingsplan kan bestaan uit workouts gemaakt door de gebruiker zelf of door andere gebruikers
- De gebruiker geeft de volgorde van de workouts in het trainingsplan aan
- Een trainingsplan heeft een naam en een beschrijving
F-06 Een gebruiker kan een lijst met trainingsplannen bekijken - De lijst laat de naam van het trainingsplan zien
- De lijst laat het aantal verschillende workouts in het trainingsplan zien
- De lijst bestaat uit de gebruikers eigen gemaakte trainingsplannen en trainingsplannen die door de gebruiker zijn toegevoegd
F-07 Een gebruiker kan een trainingsplan volgen - Een gebruiker kan zijn eigen trainingsplan volgen of het trainingsplan van een andere gebruiker
- De applicatie geeft aan wat de volgende workout in het trainingsplan is zodat de gebruiker deze makkelijk kan starten
F-08 Een gebruiker kan zijn gewicht bijhouden - Een gebruiker kan invullen wat hun huidige gewicht is
- Er kan maar een gewicht entry per dag gemaakt worden
- wanneer een gebruiker een tweede entry maakt wordt de eerste verwijderd
F-09 Gebruikers kunnen elkaar volgen - Wanneer gebruikers elkaar volgen kunnen ze workouts en trainingsplannen van elkaar gebruiken voor hun eigen trainingen
